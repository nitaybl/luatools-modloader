# Kite

> Extend LuaTools with community mods — without touching the core plugin.

## What is LuaTools?

[LuaTools](https://github.com/madoiscool/ltsteamplugin) is a Steam Millennium plugin that adds game fix management directly inside your Steam client. It lets you download and apply community game fixes (online multiplayer patches, DRM fixes, etc.) with a single click.

## What is the Mod Loader?

The Mod Loader sits **on top of** LuaTools as a lightweight extension layer. It lets you:

- 🧩 **Install mods** that add new features to LuaTools
- 🎨 **Apply themes** that change the look and feel
- 🔔 **Add notifications** for events like fix downloads
- 🔌 **Build your own mods** using a simple JavaScript API
- 💻 **Manage everything** from PowerShell

The mod loader **never modifies** core LuaTools files. Uninstalling it leaves your setup exactly as it was.
